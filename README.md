# 🚀 cURL to RestAssured Generator

Generate Java RestAssured test code from cURL commands.

## ✨ Features

- Convert cURL to RestAssured service
- Generate DTO from JSON body
- Create ready-to-run Maven project
- Clean and readable code output

---

## ⚙️ How to use

1. Open the project in IntelliJ
2. Go to `CurlGeneratorApp`
3. Paste your cURL
4. Run the application
